# ChargePay Frontend

Frontend inicial em React + TypeScript + Vite para o ecossistema ChargePay.

## Estrutura
- `src/features/auth`: telas de autenticação
- `src/features/onboarding`: fluxo de onboarding financeiro
- `src/shared`: cliente HTTP, store e utilidades compartilhadas

## Como executar
1. Instale as dependências com `npm install`
2. Inicie o ambiente com `npm run dev`
3. A aplicação ficará disponível em `http://localhost:5173`
